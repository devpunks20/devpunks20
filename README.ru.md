- **Мультирегиональный mesh**: кросс-доменный доступ к ASIC через Redis presence + федерацию пиров по gRPC

**Инфраструктура (спроектировал и развернул с нуля):**
- Кластер **Docker Swarm** на 7 нод в 3 регионах (COM/RU/US), размещение по сайтам, **zero-downtime** rolling-обновления (start-first)
- **PostgreSQL HA**: Patroni + etcd + HAProxy + PgBouncer на каждой ноде, RO/RW-сплит (чтение с реплик), двойной режим **кластер ↔ single-master** и авто-промоут при изоляции сайта (`site-warden`)
- **GlusterFS** — общее хранилище (бинари агентов, сертификаты, конфиги) на всех нодах
- **Observability**: VictoriaMetrics + vmagent + VictoriaLogs + Vector + Grafana + vmalert + Alertmanager (по нодам, gossip HA), node-exporter / cAdvisor / postgres / redis exporters
- **Edge**: nginx-proxy + acme-companion (Let's Encrypt) + внутренний per-node nginx (gRPC/HTTP-роутинг)
- **CI/CD**: GitHub Actions + self-hosted runners на manager-ноде, приватный Docker registry, автосборка образов и выкатка сервисов
- Полный **bootstrap кластера через Ansible** (prereqs → Swarm → secrets → stack → edge)

**Результат:** отказоустойчивая мультирегиональная платформа мониторинга майнинг-ферм с real-time-дашбордом, алертами, групповыми операциями и полностью автоматизированным развёртыванием инфраструктуры.
